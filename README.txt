What is this

    This is a base code to get a customized unix shell environment.

How to use

    eval the output of "bash-init" script in your bash process.
    Example: eval "$(~/custom-shell/bash-init)"

How to Customize

    Rename and edit the "samplesubsystem" codes.
    You need to add the new subsystem to SUB_SYSTEMS variable in the "init" script.
    To know detailed way of customize, read and understand the "init" script.
